# Mod 管理

Sync the Spire 内置了 Mod 管理器，方便你安装、卸载和查看 Mod。

## 打开 Mod 管理器

在仪表盘上点击「**Mod 管理**」按钮即可打开 Mod 管理器。

![Mod 管理器](/images/v2-mod-manager.png)

## 安装 Mod

支持以下几种方式安装 Mod：

- **拖拽安装**：将 Mod 压缩包（`.zip`、`.rar`、`.7z`）或文件夹直接拖拽到管理器窗口
- **点击选择**：点击管理器顶部的安装区域，通过文件选择器选取 Mod 压缩包

安装完成后，Mod 会自动出现在列表中。

## 查看 Mod 详情

点击任意 Mod 卡片可查看详细信息：

![Mod 详情](/images/v2-mod-detail.png)

- **基本信息**：名称、版本、作者、描述
- **依赖关系**：该 Mod 依赖的前置 Mod（✓ 绿色 = 已安装，✗ 红色 = 缺失）
- **被依赖**：哪些 Mod 依赖了它（点击可跳转）
- **文件列表**：查看 Mod 包含的文件

::: warning 缺失前置 Mod
如果某个 Mod 依赖的前置 Mod 未安装，卡片上会显示红色警告标识。请确保安装所有必需的前置 Mod，否则游戏可能无法正常运行。
:::

## 卸载 Mod

在 Mod 详情面板中点击「**删除**」按钮，确认后即可卸载该 Mod。

::: danger 操作不可逆
卸载会直接删除整个 Mod 文件夹，无法撤销。如果不确定，建议先备份。
:::

## 跨分支拷贝 Mod

如果你想从其他分支拷贝某个 Mod 到当前分支：

1. 点击管理器顶部的「**从分支拷贝**」按钮
2. 在下拉框中选择源分支
3. 浏览该分支的 Mod 列表，点击目标 Mod 的「**拷贝**」按钮

![从分支拷贝](/images/v2-branch-copy.png)

如果目标 Mod 有依赖的前置 Mod 且尚未安装，会弹出提示询问是否一并拷贝。

## Mod 差异比较

在上传或拉取 Mod 时，软件会自动弹出**差异预览**，直观展示即将发生的变化：

![Mod 差异预览](/images/v2-mod-diff.png)

- 🟢 **新增**：此次操作新增的 Mod
- 🟡 **更新**：版本发生变化的 Mod（显示 `旧版本 → 新版本`）
- 🔴 **移除**：此次操作将移除的 Mod
- 无变化的 Mod 以数量汇总显示在底部

确认差异后点击继续，操作才会真正执行。

## Mod 开关

仪表盘上的「**Mod 开关**」可以快速切换 Mod 模式和原版模式：

- **开启**：连接到 Mod 仓库，使用当前分支的 Mod
- **关闭**：断开 Mod 仓库，游戏恢复无 Mod 的原版状态

Mod 文件**不会被删除**，只是暂时断开连接。重新打开开关即可恢复。
