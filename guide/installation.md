## 方式二：手动下载

前往 [GitHub Releases](https://github.com/Ruikoto/sync-the-spire/releases) 下载最新版本，解压即用，无需安装。

## 系统要求

- **操作系统：** Windows 10 / 11
- **运行时：** 无需额外安装（已内置）
- **网络：** 需要能访问 Git 远程仓库（GitHub / Gitee 等）
