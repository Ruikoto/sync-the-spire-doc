# 快速上手

::: warning 首次使用前请手动备份
备份你的存档和 Mod 文件夹。本软件经过初步测试，但多一道保险总没错。
:::

## 准备工作

使用 Sync the Spire 之前，你需要准备两样东西：

1. **一个远程 Git 仓库** — 用于存储和同步 Mod 配置
2. **Sync the Spire 客户端** — [下载安装](/guide/installation)

## 第一步：准备远程仓库

Sync the Spire 使用 Git 仓库来同步数据，你需要一个远程仓库地址。

- **自己建房**：在 Git 托管平台创建一个仓库，然后邀请队友加入
- **跟随队友**：直接使用队友提供的仓库地址，跳过创建步骤

| 平台 | 说明 | 创建指南 |
|------|------|----------|
| **Gitee** | 国内访问快，推荐国内用户首选 | [查看 Gitee 配置指南](/guide/gitee) |
| **GitHub** | 全球通用，但国内部分地区访问受限 | [查看 GitHub 配置指南](/guide/github) |
| 其他平台 | GitLab、Gitea 等理论上均可 | — |

如果你要自己创建仓库，按照对应指南完成仓库创建和认证配置。如果使用队友的仓库地址，可直接跳到下一步。

## 第二步：填写配置并初始化

打开 Sync the Spire，你会看到配置页面：

![配置页](/images/git-remote-guide/config-page.png)

1. 将仓库地址粘贴到「**远程仓库 URL**」输入框
2. 填写你的**昵称**（队友会看到这个名字）
3. 选择**认证方式**（各平台推荐的方式见对应配置指南）
4. 填写或自动查找**游戏路径**和**存档路径**
5. 点击「**保存并初始化**」

::: tip 找不到路径？
- **游戏安装路径**：Steam 中右键游戏 → **管理** → **浏览本地文件**
- **存档路径**：启动游戏后按 **`~`** 键打开控制台，输入 `open saves` 回车
:::

初始化完成后自动进入主界面，接下来就可以开始使用了。

## 第三步：开始使用

配置完成后，根据你的角色选择操作。

### 做房主 — 分享 Mod 配置

你负责维护 Mod 配置，队友从你这里同步。

![主界面](/images/Screenshot1.png)

1. 在分支名输入框中填入一个名称（简单描述一下这套 Mod 配置），点击「**创建**」
2. 像平时一样安装、调整 Mods
3. 完成后点击「**上传**」

别忘了把仓库地址分享给队友，并在仓库中[邀请他们为协作者](/guide/gitee#_3-邀请队友)。

> 更多细节见 [做房主](/guide/host)

### 跟随别人 — 使用房主的配置

你只需要拉取房主分享的配置即可。

1. 点击「**切换分支**」，在列表中选择房主的分支

   ![分支列表](/images/Screenshot2.png)

2. 可预览该分支包含的所有 Mod，确认无误后点击「**同步此分支**」

   ![Mod 预览](/images/Screenshot3.png)

3. 同步完成，直接开游戏即可

之后每次联机前，点击「**拉取**」获取房主的最新配置。

> 更多细节见 [跟随别人](/guide/join)
