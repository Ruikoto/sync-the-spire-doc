## 准备工作

使用 Sync the Spire 之前，你需要准备两样东西：

1. **一个远程 Git 仓库** — 用于存储和同步 Mod 配置
2. **Sync the Spire 客户端** — [下载安装](/guide/installation)

## 配置远程仓库

| 平台 | 说明 | 配置指南 |
|------|------|----------|
| **Gitee** | 国内访问快，推荐国内用户首选。单仓库限 500MB，单文件限 50MB | [查看 Gitee 配置指南](/guide/gitee) |
| **GitHub** | 全球通用，但国内部分地区访问受限 | [查看 GitHub 配置指南](/guide/github) |
| 其他平台 | GitLab、Gitea 等理论上均可，需自行测试 | — |

选好平台后，按照对应指南完成仓库创建和软件配置即可。

## 认证方式说明

| 方式 | 适用场景 |
|------|----------|
| **免密** | Gitee 仓库（推送时系统弹出登录窗口），或任意平台的公开仓库 |
| **HTTPS** | GitHub 等平台的私有仓库，需填写用户名和 Personal Access Token |
| **SSH** | 已配置 SSH 密钥的用户，适用于所有平台 |

## 常见问题

### 游戏安装路径怎么找？

Steam 中右键游戏 → **管理** → **浏览本地文件**。

### 存档路径怎么找？

1. 启动游戏后按 **`~`** 键（波浪号键，通常在 Esc 下方）打开控制台
2. 输入命令 `open saves` 并按回车
3. 系统将直接打开存档所在的文件夹
