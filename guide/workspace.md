# 多工作区

Sync the Spire 支持创建多个工作区，可以为不同的游戏或不同的 Mod 配置创建独立的工作区，互不干扰。

## 工作区主页

打开软件后看到的就是工作区主页，以卡片形式展示所有已创建的工作区。

![工作区主页](/images/v2-home.png)

- 每张卡片显示工作区名称、游戏类型图标和配置状态
- 绿色圆点表示已配置完成，灰色表示尚未配置
- 上方有搜索框，可按名称快速过滤

## 创建工作区

1. 点击「**新建工作区**」
2. **第一步**：选择游戏类型

   ![选择游戏类型](/images/v2-wizard-step1.png)

   目前支持的游戏：
   | 游戏 | 状态 |
   |------|------|
   | 杀戮尖塔 2 | ✅ 完整支持 |
   | 星露谷物语 | 🚧 即将支持 |
   | Minecraft | 🚧 即将支持 |
   | 通用 | ✅ 基础 Git 同步 |

3. **第二步**：输入工作区名称，点击「**创建**」

   ![输入工作区名称](/images/v2-wizard-step2.png)

创建完成后自动进入配置页面。

## 标签页管理

打开一个工作区后，它会以标签页的形式显示在顶部栏中。

- 点击标签可在不同工作区之间快速切换
- 点击标签上的 × 关闭该工作区标签（不会删除工作区）
- 最左侧的「主页」标签始终存在，点击返回工作区主页

## 管理工作区

- **重命名**：在仪表盘顶部点击工作区名称即可修改
- **删除**：在主页的工作区卡片上点击删除图标，确认后删除
- **设置**：点击卡片上的齿轮图标，进入该工作区的配置页面

::: danger 删除不可恢复
删除工作区会清除该工作区的所有本地数据（仓库、备份等）。请确认不再需要后再操作。
:::
