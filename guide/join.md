# 跟随别人 — 使用房主的 Mod 配置

作为队友，你只需要拉取房主分享的配置即可。

## 操作流程

1. 在仪表盘的「联机跟随」卡片中，点击「**切换分支**」，在列表中选择房主的分支

   ![分支列表](/images/v2-branch-list.png)

2. 进入后可预览该分支包含的所有 Mod，确认无误后点击「**同步此分支**」

   ![Mod 预览](/images/v2-branch-mods.png)

   ::: tip NSFW 提示
   如果房主的分支中包含可能的 NSFW 内容，列表中会有明显的标注提醒。
   :::

3. 同步完成，点击仪表盘上的「**启动游戏**」即可开玩

之后每次联机前，点击「**拉取**」获取房主的最新配置。拉取前也会显示 Mod 差异预览，让你清楚了解哪些 Mod 发生了变化。

![拉取差异预览](/images/v2-pull-diff.png)
