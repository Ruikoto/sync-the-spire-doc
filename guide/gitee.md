# Gitee 配置指南

本文以 Gitee 为例，介绍从创建仓库到完成同步的完整流程。

## 1. 创建远程仓库

1. 访问 [Gitee](https://gitee.com/) 注册并登录。

2. 点击右上角「**+**」→「**新建仓库**」

   ![新建仓库](/images/git-remote-guide/gitee-1.png)

3. 填写仓库名称（如 `my-sts-mods`），选择开源或私有，点击「**创建**」。
   - **开源仓库** — 所有人可见，不限制协作人数
   - **私有仓库** — 仅被邀请的人可见，免费账号最多 5 名协作者

   ![仓库信息](/images/git-remote-guide/gitee-2.png)

4. 创建成功后进入仓库主页，点击「**HTTPS**」，复制仓库地址。

   ![复制链接](/images/git-remote-guide/gitee-3.png)

## 2. 配置 Sync the Spire

### 认证方式

| 方式 | 说明 |
|------|------|
| **免密**（推荐） | 推送时系统弹出 Gitee 登录窗口，无需额外配置 |
| **SSH** | 如果你已配置 SSH 密钥，也可以使用 SSH 方式连接 |

5. 打开 Sync the Spire，将复制的仓库地址粘贴到「远程仓库 URL」输入框，填写昵称，认证方式选「**免密**」。

6. 填写或自动查找游戏路径和存档路径，点击「**保存并初始化**」。初始化时会弹出 Gitee 登录窗口，输入账号密码完成认证。

   ![配置页](/images/git-remote-guide/config-page.png)

## 3. 邀请队友

推荐每位朋友各自注册 Gitee 账号，在仓库中添加为协作者，各用各的账号。

7. 进入仓库「**管理**」→ 左侧「**仓库成员管理**」→「**添加成员**」。

   ![添加成员](/images/git-remote-guide/gitee-4.png)

8. 权限选「**开发者**」，将邀请链接发给朋友。朋友接受邀请后，再把**仓库地址**一并发给他们。

   ![协作者邀请](/images/git-remote-guide/gitee-5.png)

9. 朋友在 Sync the Spire 中填入同样的仓库地址，认证方式同样选「**免密**」，初始化完成后即可同步你的 Mod 配置。
