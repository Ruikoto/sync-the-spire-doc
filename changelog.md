# 更新日志

## v2.0.2 <Badge type="tip" text="2026-04-17" />

- 修复了Mod管理器某些情况下无法正确卸载Mod的问题
- 修复了Toast占用标题栏的问题
- Bug修复和性能优化

## v2.0.1 <Badge type="tip" text="2026-04-17" />

- **多工作区**，全新UI，支持创建多工作区，可以同时管理多个游戏的 MOD 和存档
- **Mod管理**，新增Mod管理功能，可以更方便地安装、卸载、查看Mod详情
- **Mod差异比较**，在杀戮尖塔2上传、拉取Mod时，会自动比较并显示差异，更新前后的差异更直观
- **一键启动游戏**，新增启动游戏的按钮，一键启动游戏
- **其他游戏支持**，未来将新增对星露谷物语和Minecraft的支持，用户可以在创建工作区中选择游戏类型
- **多语言支持**，新增了对多种语言的支持，用户可以在设置中选择界面语言
- Bug修复和性能优化

## v1.4.2 <Badge type="tip" text="2026-03-31" />

- [⚠️重要] 修复了应用商店版本中Config路径处理错误的问题。从应用商店全新安装（以前没用过ZIP包的版本）的用户会受到这个BUG的影响，导致完全无法使用，需要立刻进行更新。

## v1.4.1 <Badge type="tip" text="2026-03-30" />

- 新增了首次使用引导。
- 关于页面中新增使用教程的链接按钮。
- 修复了打开公告和更新日志文本中自带的链接时，无法在外部浏览器中打开的BUG。

## v1.4.0 <Badge type="tip" text="2026-03-26" />

- **MOD 列表预览**，支持在切换分支时预览 MOD 列表
- **NSFW 分支标注**，智能检测分支中是否包含 NSFW 内容，并在切换分支时进行明显标注
- **自动查找游戏和存档路径**，省得手填了，目前仅支持 Steam 平台
- 修复了公告关闭后下次会再次弹出的 BUG
- 修复了某些情况下 Git 组件报认证失败的 BUG
- 替换图标库资源

## v1.3.0 <Badge type="tip" text="2026-03-25" />

- **改进更新检查和自动更新机制**，商店版本现在直接调用官方 API 进行应用内更新
- **优化游戏/MOD 文件夹选择功能**，在某些选错路径层级的情况可以自动纠错
- **优化 Toast 提示**，较长的 Toast 支持滚动查看，而非直接截断
- **加入日志系统**，提供更详细的操作记录和错误信息，方便排查问题
- **改进应用内公告**，支持 Markdown 渲染

## v1.2.3 <Badge type="tip" text="2026-03-24" />

- **上架 Microsoft Store**，提供更便捷的安装与更新方式

## v1.2.2 <Badge type="tip" text="2026-03-23" />

- **重构 UI 界面**，提供更简洁易用的视觉与交互体验
- **重做刷新逻辑**，点击刷新即可一键检测本地未提交的改动及云端未拉取的更新
- **新增昵称设置**，用于 MOD 上传至远程仓库时的作者署名
- **优化更新日志显示**，全面支持多行文本展示与 Markdown 格式渲染

## v1.2.1 <Badge type="tip" text="2026-03-20" />

## v1.2.0 <Badge type="tip" text="2026-03-20" />

- **新增存档重定向**，彻底替代此前存在缺陷的存档合并功能

- **内置 MinGit 环境**，修复并完善对 Gitee 平台的支持
