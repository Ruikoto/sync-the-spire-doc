# 常见问题

## 没装过 Git 能用吗？

能。软件会自动下载所需的 Git 组件，无需手动安装。

## 会影响游戏本体文件吗？

不会。软件只操作 Mods 文件夹和存档文件夹，不修改游戏本体。关闭 Mod 开关即可恢复原版状态。

## 我的账号密码安全吗？

认证信息使用 Windows 系统级加密（DPAPI）存储，不上传到任何服务器，仅本机当前用户可读取。

## 支持哪些 Git 平台？

支持 Gitee、GitHub、GitLab 等主流平台。Gitee 推荐使用「免密」方式，GitHub 推荐使用「HTTPS + Token」方式。详见[快速上手](/guide/getting-started)。
